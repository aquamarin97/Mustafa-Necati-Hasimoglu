### 🚀 Featured Projects

#### [PharmaLyser SMA](https://github.com/aquamarin97/pharmalyser-sma)
Production-grade desktop application for real-time PCR analysis in SMA carrier screening.
Built with Python/PyQt5 — features automated 96-well plate processing, statistical classification (KMeans + L-BFGS-B optimization), regression-based QC, and interactive data visualization at 60+ FPS.

MSc thesis project: Structure-based drug design targeting antibiotic-resistant *N. gonorrhoeae*. Screened ~4.5M compounds, performed 100 ns MD simulations, and identified novel DXR inhibitors. International collaboration (Germany × Thailand × Turkey).
